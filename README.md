# Baby PC Mode

リモートワーク中に赤ちゃんがキーボードに来た瞬間、すぐ「触って楽しい画面」に切り替えるためのミニマムなプロダクトです。

## このプロダクトの勝ち筋

「親が困る瞬間」に対して、`1クリックですぐ効く` のが価値です。

- 赤ちゃんがキーボードを触っても怒られず、むしろ反応が返ってくる
- 親は数分だけでも打ち合わせや思考をつなげやすい
- 説明なしでも体験がわかるので、SNSや友人紹介で広がりやすい

## 初版で入れている価値

- 大きなボタンひとつで `赤ちゃんモード` に切り替え
- 可能ならブラウザの全画面表示に入り、赤ちゃんに渡しやすい
- キーボード入力やタップごとに、色・ことば・図形が楽しく反応
- Vercel にそのまま載せやすい `Next.js` 構成

## ローカル起動

```bash
npm install
npm run dev
```

## 公開の最短ルート

1. GitHub に新規リポジトリを作る
2. このプロジェクトを push する
3. Vercel で GitHub リポジトリを import する
4. `main` or `master` ブランチをデプロイする

## Cursor で進めるときのおすすめ運用

1. `Cursor` でこのフォルダを開く
2. 画面・文言・導線を AI に一緒に詰めてもらう
3. 変更を細かく commit して `GitHub` に push する
4. `Vercel` の Preview Deployments でスマホ表示も毎回確認する

## Supabase を入れるタイミング

初版では不要です。以下が必要になったら導入するとちょうど良いです。

- 保護者アカウント
- 子どもごとのお気に入りテーマ保存
- 利用回数や継続率の計測
- 管理画面やA/Bテスト

## 次に足すと良いもの

- 効果音のオンオフ
- 赤ちゃんごとのテーマ切替
- 一定時間で自動終了
- 利用ログ保存
- 保護者向け設定画面

`Supabase` は認証やログ保存が必要になったタイミングで導入すれば十分です。
